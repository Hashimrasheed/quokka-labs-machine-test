# quokka-labs-machine-test

### Project setup

1. Clone the repo
 ```sh
 git clone git@github.com:Hashimrasheed/quokka-labs-machine-test.git
 ```
2. For installing packages
  ```sh
  npm install
  ```
3. run application
  ```sh
  npm run start
  ```

### Technologies
Node.Js, Express.Js, MongoDB, JWT,
